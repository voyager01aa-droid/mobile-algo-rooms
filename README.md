# Mobile Algorooms Clone
This repository contains a mobile-friendly, single-user algorithmic trading platform built specifically for Streamlit Cloud and Kotak Neo API.

## Setup Instructions for Mobile Users
1. Copy the code from `requirements.txt`, `kotak_neo_engine.py`, and `app.py`.
2. Go to GitHub on your mobile web browser and paste them into a private repository.
3. Link the repository to [Streamlit Community Cloud](https://share.streamlit.io) via mobile browser.
4. Access your dynamic trading panel instantly from your phone!
